# 🏏 BPL Dream 11

A fantasy cricket web app for Bangladesh Premier League fans — build your dream team, manage coins, and explore player stats through a sleek, responsive interface.

🔗 **Live Demo:** https://bpl-dream-11-demo.netlify.app/

## ✨ Features

- 🪙 **Coin System** — Claim free credits to start building your squad
- 🏏 **Player Selection** — Pick players by role and price within budget
- ❌ **Player Removal** — Easily drop players from your active lineup
- 🔄 **Real-time UI** — Instant updates powered by React state management
- 🔔 **Toast Notifications** — Clear success/error feedback on every action
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v16 or higher)
- [Git](https://git-scm.com/)

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/SAIEED12/BPL-Dream-11.git
   cd BPL-Dream-11
```

2. **Install dependencies**
```bash
   npm install
```

3. **Start the development server**
```bash
   npm run dev
```

4. **Open in your browser**
```
   http://localhost:5173
```

### Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder, ready to deploy.
